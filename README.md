# 声明 #

- 使用本项目即*为清楚明白以下声明*，本人**不会对此另行解释**
- bot能获取到的信息包括：
```
- 用户：
	- username
	- firstname
	- lastname
	- userid
	- language_code
	- text（带有命令的）
- 群组：
	- group_type
	- group_id
	- group_name
	- group_info
```
